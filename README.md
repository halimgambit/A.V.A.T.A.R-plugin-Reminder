# Reminder
Creer des rappels vocal avec AVATAR

This plugin is an add-on for the [A.V.A.T.A.R](https://avatar-home-automation.github.io/docs) framework.

🎯 Usage
Commandes :

- Rappel moi de partir dans une heure.
- Rappel moi de d'appeler paul dans 15 minutes.

# Multi-room
The Reminder plugin is fully multi-room.

# Multi-language
The Reminder plugin relies solely on the system's available languages.
